rundyfi.py¶

Create DYFI products for a given event.

usage: rundyfi.py [-h] [--push] [--redo] [--configfile CONFIGFILE] evid

Positional Arguments¶

evid Event ID

Named Arguments¶

--push

Push event products to PDL (not yet implemented)

Default: False

--redo

Redo all products even if they exist

Default: False

--configfile

Specify config file

Default: "./config.yml"

makemovie.py¶

Create movie frames for a given event. This runs successive processes of DYFI on larger and larger timeslices of data. The output is an APNG file (default ./movie.apng).

usage: app/makemovie.py [-h] [--tmax [TMAX]] [--framelength [FRAMELENGTH]]
                        [--type [TYPE]] [--output OUTPUT]
                        evid

Positional Arguments¶

evid Event ID

Named Arguments¶

--tmax

Length of movie in minutes

Default: 60

--framelength

Time of each frame, in minutes

Default: 1

--type

Type of aggregation to display

Default: "geo_10km"

--output

Output filename

Default: "movie.apng"

makescreenshot.py¶

Create a static image .png file for a given event. The event data must already exist; run rundyfi.py first. The output is a PNG file (default ./screenshot.png).

usage: app/makescreenshot.py [-h] [--configfile CONFIGFILE] input [output]

Positional Arguments¶

input GeoJSON file
output

output filename

Default: "screenshot.png"

Named Arguments¶

--configfile

Specify config file

Default: "./config.yml"
